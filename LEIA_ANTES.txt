LEIA ANTES — Controle BAR v1.5.0

Este ZIP é o projeto Android nativo para subir no GitHub e gerar o APK pelo GitHub Actions.

Não é imagem. Não é mockup. É o projeto fonte completo.

PIN inicial do gestor: *177
Senha PRO: *177

Na primeira configuração, defina o celular do gestor como GESTOR digitando o PIN.
Os celulares de atendimento devem ficar como ATENDENTE.

Atendente não consegue virar gestor sem o PIN administrativo.
